# Interactive Resume App

This workspace contains a static interactive resume experience for Yaswanth Anumolu.

## Run locally

Use either command from `C:\Users\anumo\OneDrive\Desktop\ResumeBuild`:

```powershell
python -m http.server 4173
```

or

```powershell
py -m http.server 4173
```

Then open [http://localhost:4173](http://localhost:4173).

## Update content

Edit `data.js` to replace placeholder contact links and refine experience, projects, education, certifications, and achievements without touching the UI code.
